# OpenEnvX

**The local-first development runtime for micro-SaaS builders.**

Built by developers, for developers. Zero friction. Zero lock-in.

> ⚠️ **Early Development**: This tool is currently in early development. Features and APIs may change.

---

## What is OpenEnvX?

OpenEnvX is a **local development foundation** for modern micro-SaaS applications.

### Coming Soon

- **One command to rule them all** - `openenvx up` starts your entire stack (Postgres, Redis, MinIO, Qdrant...)
- **Opinionated SaaS foundation** - Next.js + Drizzle + BetterAuth + Stripe, pre-wired
- **Unified workflow** - From `openenvx init` to deployment

---

## Philosophy

**OpenEnvX is not:**

- A PaaS or deployment platform
- Kubernetes replacement
- Serverless framework

**OpenEnvX is:**

- A developer productivity tool
- Your local development foundation
- A workflow, not just code
